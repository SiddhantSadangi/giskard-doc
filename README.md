## Setup

```batch
git clone https://github.com/Giskard-AI/giskard.git
cd giskard
docker-compose up -d
```

Yes, that's all!  Then start inspecting & testing your ML models at [**http://localhost:19000/**](http://localhost:19000/)****

{% hint style="info" %}
**login**: admin  **password**: admin
